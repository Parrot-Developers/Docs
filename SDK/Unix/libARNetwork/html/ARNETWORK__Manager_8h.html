enum eARNETWORK_MANAGER_CALLBACK_RETURN

Enumerator
ARNETWORK_MANAGER_CALLBACK_RETURN_DEFAULT 

default value must be returned when the status callback differ of ARNETWORK_MANAGER_CALLBACK_STATUS_TIMEOUT

ARNETWORK_MANAGER_CALLBACK_RETURN_DATA_POP 

pop the data (default behavior)

ARNETWORK_MANAGER_CALLBACK_RETURN_RETRY 

reset the number of retry

ARNETWORK_MANAGER_CALLBACK_RETURN_FLUSH 

flush all input buffers

enum eARNETWORK_MANAGER_CALLBACK_STATUS

status sent by the callback.

Enumerator
ARNETWORK_MANAGER_CALLBACK_STATUS_SENT 

data sent

ARNETWORK_MANAGER_CALLBACK_STATUS_ACK_RECEIVED 

acknowledged received

ARNETWORK_MANAGER_CALLBACK_STATUS_TIMEOUT 

timeout occurred, data not received; the callback must return what the network manager must do with the data.

ARNETWORK_MANAGER_CALLBACK_STATUS_CANCEL 

data will not sent

ARNETWORK_MANAGER_CALLBACK_STATUS_FREE 

free the data sent without Data Copy.

ARNETWORK_MANAGER_CALLBACK_STATUS_DONE 

the use of the data is done, the date will not more used

Create a new Manager.

Warning
This function allocate memory
Postcondition
ARNETWORKAL_Manager_t must be created to initialize the network.
ARNETWORK_Manager_Delete() must be called to delete the Network and free the memory allocated.
Parameters
[in]networkALManagerAn initialized ARNETWORKAL_Manager_t instance which will be used for all network operations
[in]numberOfInputNumber of input buffer
[in]inputParamArrarray of the parameters of creation of the inputs. The array must contain as many parameters as the number of input buffer.
[in]numberOfOutputNumber of output buffer
[in]outputParamArrarray of the parameters of creation of the outputs. The array must contain as many parameters as the number of output buffer.
[in]pingDelayMsMinimum amount of time (ms) between two pings. Put a negative value to disable ping, and zero to use default value
[in]onDisconnectCallbackfuction called on disconnect.
[in]customDatacustom Data sent to the callbacks.
[out]errorerror output.
Returns
the new Manager
Note
This creator adds for all output, one other IOBuffer for storing the acknowledgment to return. These new buffers are added in the input and output buffer arrays.
Warning
The identifiers of the IoBuffer should not exceed the value 128.
