ARSAL_Print_PrintRaw
int ARSAL_Print_PrintRaw(eARSAL_PRINT_LEVEL level, const char *tag, const char *format,...)
Prints a formatted output.

Prints a specific output.

This is the default entry point for printing. This function outputs a string of the following format: [LEVEL] TAG | HH:MM:SS:mmm | function:line - FORMAT

Parameters
levelThe print level (eARSAL_PRINT_LEVEL enum)
tagA short tag which will prefix the log timestamp
formatThe format string to print
...The format parameters
