libARDataTransfer Documentation

Introduction

The ARDataTransfer library provides a way to retrieve device data and medias files stored in a device.

Manager

Header file

The ARDataTransfer Manager is the main interface to that allows to initialize the library an get access to the two others sub-modules: DataDownloader and MediasDownloader.

Submodules

Here is a short description of the two libARDataTransfer submodules: DataDownloader and MediasDownloader.

Data Downloader

Header file

This submodule implements a mechanism to download automatically the devive data (flights or rides) through an FTP device server.

Medias Downloader

Header file

This submodule implements firstly a mechanism to retrieve the available Medias (Photos and Videos) list on a device through an FTP device server. It implements also a queue process to download user chosen medias via FTP.
